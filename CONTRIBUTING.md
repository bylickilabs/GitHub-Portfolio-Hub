   ```bash

   ```
2. Commit your changes with a clear message:
   ```bash
   git commit -m "Add: my new feature"
   ```
3. Push to your fork and submit a pull request.
4. Ensure your PR:

### Code Style
- Use **clear and consistent naming** for variables and functions.
